# Sample Hardhat Project

This project demonstrates a basic Hardhat use case. It comes with a sample contract, a test for that contract, and a Hardhat Ignition module that deploys that contract.

Try running some of the following tasks:

```shell
npx hardhat help
npx hardhat test
REPORT_GAS=true npx hardhat test
npx hardhat node
npx hardhat ignition deploy ./ignition/modules/Lock.js

Документация Voting App
Обзор проекта
Voting App — это децентрализованное приложение (dApp), позволяющее пользователям участвовать в безопасном и прозрачном голосовании на блокчейне Ethereum (тестовая сеть Sepolia). Оно состоит из:

Смарт-контракта: управляет логикой голосования и хранит результаты на блокчейне

Фронтенда на React: предоставляет пользовательский интерфейс для голосования

Интеграции с MetaMask: обеспечивает подключение кошельков и проведение транзакций

Ключевые функции
Безопасное голосование: каждый адрес кошелька может проголосовать только один раз

Ограниченное по времени голосование: период голосования задаётся при развертывании контракта

Результаты в реальном времени: количество голосов обновляется автоматически

Прозрачность: все голоса неизменно записываются в блокчейн

Панель администратора: владелец контракта может добавлять кандидатов
```
